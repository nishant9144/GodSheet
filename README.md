## Installation

To install and run this project, follow these steps:

1. Clone the repository:
    ```sh
    git clone https://github.com/nishant9144/Spreadsheet.git
    ```
2. Navigate to the project directory:
    ```sh
    cd Spreadsheet
    ```
3. Build the project using Makefile:
    ```sh
    make
    ```
